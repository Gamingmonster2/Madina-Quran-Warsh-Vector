# Madina Quran - Warsh Recitation (Vector Graphics)

هذا المستودع مخصص لأرشفة وحفظ النسخة الرقمية الأصلية لمصحف المدينة النبوية برواية ورش عن نافع بصيغة المتجهات (AI).

## التفاصيل التقنية (Technical Details):
- **الرواية (Recitation):** ورش عن نافع (Warsh).
- **نوع الملف (File Type):** متجهات (Vector Graphics - AI).
- **المصدر (Source):** مجمع الملك فهد لطباعة المصحف الشريف - نسخة مؤرشفة (King Fahd Complex).
- **الهدف (Goal):** توفير النسخة للمطورين والمصممين لاستخدامها في التطبيقات والطباعة بجودة عالية (Infinite Scaling).
